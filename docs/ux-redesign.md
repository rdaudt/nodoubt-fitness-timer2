## Support to app modes
Coach Mode default value is ON.
Users shall be able to set Coach Mode ON/OFF at any time. It shall be done in the Home Page, at the top of the user's Timer's list.
## Impact on UI
When Coach Mode is OFF, it is assumed that the user is doing HIIT by itself and the concept of Stations doesn't apply. The app shall surface in the UI the correct expressions according to the table below.

| Coach Mode ON           | Coach Mode OFF      |
| ----------------------- | ------------------- |
| Number of Stations      | Number of Sets      |
| Station Transition Time | Set Transition Time |

The Timer's attribute Start Station Work Manually is only shown for the user when the app is in Coach Mode. Its default is OFF.

## Timer Editor
We shall switch from a graphical way to configure Timers to one based on attributes typed in by the user. No more quick-add tools should be used. Still, Timer deletion shall support sliding timers to the left.

## Themes
Do not changed the UI themes as per this re-design.
